# Global Conflict Cost Tracker

A live estimator tracking the costs of active global conflicts — military expenditure, death tolls, supply chain disruptions, and aid flows.

## Conflicts Tracked

- **2026 Iran War** (Feb 28, 2026 — Present): US, Israel, Iran, Gulf States expenditure with death tolls, Strait of Hormuz impact, and US aid to Israel tracking
- **Russia-Ukraine War** (Feb 24, 2022 — Present): Russia and Ukraine expenditure with casualties, Western aid flows, and economic impact

## Live Site

[View the tracker](https://www.perplexity.ai/computer/a/global-conflict-cost-tracker-L0aBczrETpWjImSbAHtD_g)

## Tech Stack

- Vanilla HTML/CSS/JS (no build tools)
- JetBrains Mono + Inter typography
- Dark/light mode toggle
- Mobile responsive

## Data Sources

All citations link to primary sources including Al Jazeera, Reuters, BBC, Mediazona, Hengaw, HRANA, UN OHCHR, NPR, Moscow Times, and others. No Wikipedia links — only direct primary source citations.
